## thief.py
```
n个宝物，每个宝物有自己的重量与价值
m容量的背包，尝试在有限的容量内，装入最大价值的宝物
先输入n 与 m
再输入宝物信息
```
```
例如输入
宝物数量：3
背包容量：12
重量与价值（空格隔开）：5 5
重量与价值（空格隔开）：6 5 
重量与价值（空格隔开）：4 9
输出最大价值：14
```
